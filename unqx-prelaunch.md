## РАЗДЕЛ 2 — Требования App Store (Apple)

### 2.1 Обязательные разрешения с объяснением

Apple отклоняет если:
- Запрашиваешь разрешение без объяснения
- Объяснение не конкретное ("нужен доступ к камере" — плохо)

Проверить каждый NSUsageDescription в infoPlist.
Текст должен объяснять КОНКРЕТНУЮ пользу для пользователя.

### 2.2 NFC — особые требования Apple

- Entitlement com.apple.developer.nfc.readersession.formats обязателен
- В App Store Connect нужно выбрать "NFC Tag Reading" capability
- Описание использования NFC должно быть в App Review Notes

Добавить в поле "Notes for App Review" при сабмите:
```
NFC используется для:
1. Чтения NFC-меток других пользователей UNQX (тип NTAG213)
2. Записи ссылки unqx.uz/SLUG на NFC-метку пользователя
3. Проверки состояния NFC-метки

Для тестирования NFC функции используйте физическое устройство
с NFC поддержкой. Тест-аккаунт предоставлен ниже.
```

### 2.3 In-App Purchase vs внешние платежи

КРИТИЧНО: Apple не разрешает ссылки на внешние платёжные системы
(Payme, Click) внутри iOS приложения.

Варианты:
А) Убрать кнопку оплаты из iOS приложения полностью.
   Добавить текст: "Оформите подписку на unqx.uz"

Б) Реализовать через Apple In-App Purchase (долго и сложно)

- Зачем: сканирование QR-кодов
